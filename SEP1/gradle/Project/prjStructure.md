# Project Structure

**Gradle Structure**
```
build.gradle.kts (Kotlin DSL)
settings.gradle.kts (Project setting)
./gradle (Gradle wrapper)
gradlew (run script Gradle wrapper for Linux)
gradlew.bat (run script Gradle wrapper for windows)

*build (build from Gradle)
```

**Java Structure**
```
src/main/java/
-> Package: com.example
-> Main: Main.java
=> com.example.Main

src/test/java/
-> Package: com.example
-> Main: Main.java
=> com.example.Main
```
